================================================================================
STANDARDIZED COURSE FORMAT TEMPLATE
For All HTML Course Pages (Selenium, Django, SQL, Python, etc.)
================================================================================

PROJECT OVERVIEW:
This template ensures all course pages follow a consistent, professional structure
with interactive learning features, code examples, quizzes, and responsive design.

================================================================================
PAGE STRUCTURE & LAYOUT
================================================================================

1. FIXED HEADER (Top of Page)
   ├─ Logo with icon (emoji) and course name with accent color
   ├─ Progress bar showing completion status
   └─ Format: "X / Total Days" counter

2. SIDEBAR NAVIGATION (Left Side, Fixed)
   ├─ "10-Day Course" section header
   ├─ Day buttons (numbered 01-10 with labels)
   ├─ Visual indicators:
   │  ├─ Day number in numbered badge
   │  ├─ Active state highlighting (purple border-left + bg)
   │  ├─ Completed state (green checkmark icon)
   │  └─ Hover effects for all buttons
   └─ Responsive: Collapses on tablets (768px breakpoint)

3. MAIN CONTENT AREA (Center, Scrollable)
   ├─ Day panels (hidden by default, one active at a time)
   ├─ Animated fade-in transitions between days
   └─ Padding and max-width for readability

================================================================================
EACH DAY PANEL STRUCTURE
================================================================================

DAY HERO SECTION (Top of Day Panel)
├─ Background: Gradient (purple to green with 15% opacity)
├─ Border: 1px solid with custom border color
├─ Content:
│  ├─ Day Tag: "DAY 01 / 10" (small, accent green)
│  ├─ Main Heading: Large title with span for accent color
│  ├─ Description: Gray subtext explaining day's focus
│  └─ Meta Information:
│     ├─ Difficulty level (Beginner/Intermediate/Advanced) - green dot
│     ├─ Topic category - purple dot
│     └─ Estimated duration (e.g., ~2 Hours) - orange dot
└─ Decorative: Radial gradient overlay (top-right corner)

TABBED INTERFACE (Below Hero)
├─ 3 Tabs per day:
│  ├─ 📖 Theory
│  ├─ 💻 Code
│  └─ 🧠 Quiz
├─ Tab styling:
│  ├─ Inactive: Gray text, transparent background
│  ├─ Active: Primary color (purple) text + bottom border
│  └─ Hover: Text color change to white
└─ Tab panes: Display block only when active, fade-in animation

================================================================================
JAVASCRIPT FUNCTIONALITY
================================================================================

MAIN FUNCTIONS:

1. showDay(dayNumber)
   ├─ Hides all day panels
   ├─ Shows selected day panel with fade-in
   ├─ Updates sidebar active state
   ├─ Updates progress bar
   └─ Resets all tabs to Theory (first tab)

2. switchTab(buttonElement, tabPaneId)
   ├─ Removes active class from all tabs/panes
   ├─ Adds active class to clicked tab and corresponding pane
   ├─ Triggers fade-in animation
   └─ Maintains tab state per day

3. checkAns(buttonElement, correctOrWrong, feedbackMessage)
   ├─ Adds correct/wrong class styling
   ├─ Disables all option buttons in the quiz card
   ├─ Displays feedback message with appropriate styling
   └─ Prevents further interaction with that question

4. copyCode(buttonElement)
   ├─ Finds parent code-wrap
   ├─ Extracts <code> text content
   ├─ Copies to clipboard
   ├─ Changes button text and styling to "Copied!"
   ├─ Reverts after 2000ms
   └─ Provides user feedback

5. markComplete(dayNumber)
   ├─ Marks day as complete
   ├─ Updates progress bar and counter
   ├─ Changes button styling to completed state
   ├─ Updates sidebar day button to show completion checkmark
   └─ Persists state (localStorage recommended)

ANIMATION KEYFRAMES:
└─ fadeIn:
   ├─ from: opacity 0, translateY 10px
   └─ to: opacity 1, translateY 0
   ├─ Duration: 0.3s ease
   └─ Applied to day panels and tab panes
