Database Basics
🏗️ Database Structure
A database contains multiple tables, each with columns (attributes) and rows (records).
  • Database: Container for tables and related data
  • Table: Collection of related records organized in rows and columns
  • Column: Represents an attribute with a specific data type
  • Row: Single record containing values for all columns
  • Primary Key: Unique identifier for each row in a table
  • Foreign Key: Links rows in one table to rows in another table

Basic SQL Syntax
SQL
-- CREATE a new database
CREATE DATABASE company;

-- CREATE a table with columns
CREATE TABLE employees (
  id INT PRIMARY KEY,
  name VARCHAR(100),
  salary DECIMAL(10,2),
  department VARCHAR(50)
);

-- SELECT all records from a table
SELECT * FROM employees;
Common Data Types
  • INT - Integer numbers (whole numbers)
  • VARCHAR(n) - Variable length text up to n characters
  • DECIMAL(p,s) - Fixed-point decimal numbers
  • DATE - Store dates (YYYY-MM-DD)
  • DATETIME - Store date and time
  • BOOLEAN - True or False values
