        • ▶Empty search shows message
        • ▶Special characters handled
        • ▶Search suggestions / autocomplete
        • ▶Filters and sorting work

        • ▶Add / remove items correctly
        • ▶Quantity update reflects price

        • ▶Checkout flow completes
        • ▶Payment gateway integrates

        • ▶Required field validation

        • ▶Success message on submit
        • ▶Data received at admin email

        • ▶Allowed file types accepted

        • ▶File size limit enforced
        • ▶Uploaded file displays correctly

      • ▶2.1 Layout & Alignment
        • ▶No overlapping elements
        • ▶Consistent margins and padding
        • ▶Grid alignment is clean
        • ▶Footer sticks to bottom
      • ▶2.2 Typography
        • ▶Font family consistent throughout
        • ▶Heading hierarchy (H1→H2→H3) correct
        • ▶Font size readable on all screens
        • ▶Line height not too cramped
      • ▶2.3 Colors & Branding
        • ▶Brand colors used consistently
        • ▶Sufficient contrast for readability
        • ▶No random color inconsistencies
      • ▶2.4 Images & Icons
        • ▶All images load without broken icon
        • ▶Images not stretched or distorted
        • ▶Icons are pixel-perfect and consistent
        • ▶Favicon present in browser tab
      • ▶2.5 Interactive Elements
        • ▶Hover effects on buttons / links
        • ▶Focus state visible on keyboard nav
        • ▶Tooltips show correct info
        • ▶Modals / popups open and close
        • ▶Sliders and carousels work
        • ▶Accordions expand / collapse
    • ▶ 3. Responsiveness Testing All Devices
      • ▶3.1 Breakpoints
        • ▶Desktop — 1920px, 1440px, 1280px
        • ▶Laptop — 1366px, 1024px
        • ▶Tablet — 768px, 820px
        • ▶Mobile — 375px, 390px, 414px
      • ▶3.2 Layout Checks
        • ▶No horizontal scrollbar on mobile
        • ▶Text doesn't overflow containers
        • ▶Images scale proportionally
        • ▶Columns stack correctly on small screens
      • ▶3.3 Mobile-Specific
        • ▶Hamburger menu opens / closes
        • ▶Touch targets ≥ 44px (tap-friendly)
        • ▶Phone numbers are tap-to-call
        • ▶Keyboard doesn't break layout on focus
    • ▶ 4. Link Testing Navigation
      • ▶4.1 Internal Links
        • ▶Menu links navigate correctly
        • ▶CTA buttons go to right page
        • ▶Anchor links scroll to section
      • ▶4.2 External Links
        • ▶Open in new tab (target="_blank")
        • ▶Destination URL is correct
        • ▶No 404 errors on destination
      • ▶4.3 Broken / Redirect Links
        • ▶404 pages show friendly error
        • ▶No infinite redirect loops
        • ▶Redirect lands on correct page
      • ▶4.4 Special Links
        • ▶mailto: opens email client
        • ▶tel: opens dialer on mobile
        • ▶PDF / file download links work
    • ▶ 5. Form Testing Input Validation
      • ▶5.1 Validation
        • ▶Required fields show error if empty
        • ▶Email format validated
        • ▶Phone number format checked
        • ▶Min / max character limits enforced
        • ▶Numeric fields reject alphabets
      • ▶5.2 Boundary Value Testing
        • ▶Exactly at min limit (e.g. 8 chars password)
        • ▶One below min — should fail
        • ▶One above max — should fail
        • ▶Exactly at max limit — should pass
      • ▶5.3 Negative Testing
        • ▶SQL injection in fields
        • ▶Script tags / XSS attempt
        • ▶Extremely long text input
        • ▶Pasting special / Unicode characters
      • ▶5.4 Submission & UX
        • ▶Success message after submit
        • ▶Double submit prevention
        • ▶Tab order is logical
        • ▶Auto-fill works correctly
        • ▶CAPTCHA doesn't block real users
    • ▶ 6. Cross-Browser Testing Compatibility
      • ▶6.1 Browser Coverage
        • ▶Google Chrome (latest)
        • ▶Mozilla Firefox (latest)
        • ▶Microsoft Edge
        • ▶Safari (macOS / iOS)
        • ▶Opera / Brave (optional)
      • ▶6.2 What to Check Per Browser
        • ▶CSS layout renders identically
        • ▶Fonts don't substitute
        • ▶JavaScript features work
        • ▶Forms validate and submit

        • ▶Console has no JS errors (F12)

        • ▶Network tab shows no failed requests
        • ▶Lazy loading works on images

        • ▶No jank / lag while scrolling

        • ▶Animations don't freeze the page
